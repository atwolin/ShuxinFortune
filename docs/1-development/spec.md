# 專案規格書：舒心好運籤 (Shuxin-Fortune)

1. 專案概述 (Project Overview)

    - 專案名稱：舒心好運籤 shuxin-fortune
    - 核心目標：一個可愛風格的手機版抽籤網頁，透過隨機抽籤給予學生鼓勵 。
    - 首頁標語：給正在努力的你：你不孤單 。

2. 視覺與使用者體驗設計 (UI/UX Design)

    - 整體風格：
        - 風格：Q版台灣廟宇造型，走溫柔可愛路線 。
        - 色系：以紅色與黃色為暖色主色調 。
        - 視覺特效：要有金色光暈效果 。
        - 動畫互動：
            - 搖籤動作：點擊籤筒後，籤筒會搖動 。
            - 出籤效果：籤會真的從籤筒中「往上滑出來」 。
            - 裝飾動畫：抽籤時會有星星飛出來的特效 ，以及淡入動畫 。
    - 音效 (Audio)：
        - 配有輕柔不吵雜的搖籤聲 。

3. 功能需求 (Functional Requirements)

    A. 學生端
        - 裝置支援：手機友善 。
        - 使用限制：不限抽籤次數，可重複抽取 。
        - 抽籤邏輯：隨機顯示一句籤詩 。
    B. 老師端 / 管理功能
        - 編輯功能：
            可以直接修改籤詩內容 。
            無需撰寫程式碼 (No-code) 。
            支援在手機或電腦上直接修改 。
        - 資料儲存：
        修改後的內容會自動儲存。

4. 資料內容 (Data Content)
    預設籤詩資料庫：
    提供不同類型的籤詩，例如給學測前、給畢業時

5. 交付與部署 (Deliverables & Deployment)
    - 網址：
        提供專屬網址，可長期使用及分享 。
